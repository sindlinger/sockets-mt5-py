# MT5 Hotkeys (referencia)

Lista de atalhos do MetaTrader 5. **Estes atalhos sao da UI do terminal** e
nao sao acionados pelo servico MQL. Servem apenas como consulta.

## Chart Window
- Left Arrow: rolar grafico para esquerda
- Right Arrow: rolar grafico para direita
- Up Arrow: rolar rapido para esquerda (ou subir se escala fixa)
- Down Arrow: rolar rapido para direita (ou descer se escala fixa)
- NumPad 5: restaurar escala vertical automatica
- Page Up: rolar rapido para esquerda
- Page Down: rolar rapido para direita
- Home: ir para o inicio do grafico
- End: ir para o fim do grafico
- - : zoom out
- + : zoom in
- Delete: deletar objetos selecionados
- Backspace: deletar ultimo objeto
- Enter: abrir/fechar barra de navegacao rapida

## Charts (independente da janela ativa)
- F2: abrir Task Manager
- F7: propriedades do EA anexado ao grafico
- F8: propriedades do grafico
- F12: mover 1 barra para esquerda
- Shift+F12: mover 1 barra para direita
- Shift+F5: perfil anterior
- Alt+1: barras
- Alt+2: candles
- Alt+3: linha
- Alt+W: janela de gerenciar charts
- Alt+Backspace / Ctrl+Z: desfazer delecao de objeto
- Ctrl+A: ajustar altura das janelas de indicadores
- Ctrl+B: lista de objetos
- Ctrl+F: crosshair
- Ctrl+G: mostrar/ocultar grid
- Ctrl+H: mostrar/ocultar OHLC
- Ctrl+I: lista de indicadores
- Ctrl+K: mostrar/ocultar volumes reais
- Ctrl+L: mostrar/ocultar volumes
- Ctrl+P: imprimir grafico
- Ctrl+S: salvar como CSV/PRN/HTM
- Ctrl+W / Ctrl+F4: fechar grafico atual
- Ctrl+Y: mostrar/ocultar separadores de periodo
- Ctrl+F5: proximo perfil
- Ctrl+F6: grafico anterior
- Ctrl+Shift+F6: proximo grafico

## Market Watch
- F9: nova ordem
- Space/Tab: alternar abas (Symbols/Details/Trading/Ticks)
- A: auto ajustar colunas
- G: mostrar/ocultar grid

## Navigator
- Enter: abrir EA/indicador/script (ou autorizar conta)
- Insert: abrir nova conta (Accounts)
- Delete: deletar conta/EA/indicador/script
- G: mostrar/ocultar grid (Favorites)

## Data Window
- Ctrl+C: copiar para clipboard
- A: auto sizing de colunas
- G: mostrar/ocultar grid

## Toolbox
- F9: nova ordem
- Enter: abrir item selecionado (depende da aba)
- Insert: criar item (depende da aba)
- Delete: apagar item (depende da aba)
- Space: expandir mail/alerta (depende da aba)
- C: copiar linha do journal
- D: baixar app (Code Base)
- R: mostrar/ocultar categorias de news
- A: auto sizing de colunas
- G: mostrar/ocultar grid

## Acoes comuns
- Esc: fechar dialogos
- F1: abrir Userguide
- F3: abrir variaveis globais
- F4: abrir MetaEditor
- F6: abrir Tester (EA no chart)
- F9: nova ordem
- F10: Quotes Window
- F11: fullscreen
- Alt+F4: fechar plataforma
- Ctrl+C / Ctrl+Insert: copiar
- Ctrl+D: abrir/fechar Data Window
- Ctrl+E: habilitar/disable EAs
- Ctrl+M: abrir/fechar Market Watch
- Ctrl+N: abrir/fechar Navigator
- Ctrl+O: abrir Settings
- Ctrl+R: abrir/fechar Tester
- Ctrl+T: abrir/fechar Toolbox
- Ctrl+F9: foco no Trade tab
